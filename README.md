
# 🏐 VolleySheet ( 🚧 Abandoned)

Free Volleyball Scorer

## Features

- Manage teams, rosters and line-ups
- Keeps track of rotations during match
- Enforces rules for substitutions, liberos and time outs
- Export match reports
